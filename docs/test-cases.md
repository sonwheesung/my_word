# My Word - 테스트 케이스

> 이 문서는 테스트 실행의 기준 문서입니다.
> 기능 추가/수정/삭제 시 반드시 이 문서도 함께 업데이트합니다.
> 사용자(개발자)도 직접 케이스를 추가/수정할 수 있습니다.

## 테스트 분류

| 분류 | 설명 | 실행 시점 |
|------|------|----------|
| UT | 단위 테스트 (함수/로직) | 코드 변경 시 |
| IT | 통합 테스트 (서비스↔저장소, import) | 의존성/서비스 변경 시 |
| ST | 시스템 테스트 (E2E 화면 흐름) | 화면/라우팅 변경 시 |
| FT | 퍼즈/악의적 입력 테스트 | 사용자 입력 관련 변경 시 |
| BT | 빌드 테스트 (tsc, expo) | 모든 변경 시 |
| RT | 회귀 테스트 (기존 기능) | 모든 변경 시 |
