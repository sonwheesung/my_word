개인정보처리방침

My Word (패키지명: com.myword.front)

1. 개인정보의 수집 및 이용 목적

My Word(이하 "앱")는 사용자의 단어 학습을 돕기 위한 목적으로 개발된 애플리케이션입니다. 본 앱은 별도의 회원가입 없이 사용 가능하며, 개인정보를 서버로 수집하거나 전송하지 않습니다.

2. 수집하는 개인정보 항목

본 앱은 다음과 같은 데이터를 기기 내부(로컬)에만 저장합니다:

  • 사용자가 등록한 단어, 뜻, 예문, 태그, 메모
  • 카테고리 정보
  • 퀴즈 결과 및 학습 통계
  • 앱 설정 (테마 등)

위 데이터는 모두 기기의 AsyncStorage(로컬 저장소)에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

4. 광고 서비스

본 앱은 Google AdMob을 통해 광고를 제공합니다. AdMob은 광고 제공을 위해 기기의 광고 식별자(GAID)를 수집할 수 있습니다. 이에 대한 자세한 사항은 Google 개인정보처리방침을 참고해 주세요.

5. 외부 API 사용

단어 검색 기능 사용 시 다음 외부 API에 검색어가 전송될 수 있습니다:

  • Google Translate API (번역 기능)
  • Free Dictionary API (사전 검색)

해당 API에는 검색한 단어만 전송되며, 사용자를 식별할 수 있는 정보는 포함되지 않습니다.

6. 데이터 보관 및 삭제

모든 데이터는 사용자의 기기에만 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다. 앱 내에서 개별 단어, 카테고리 등을 직접 삭제할 수도 있습니다.

7. 아동의 개인정보 보호

본 앱은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보처리방침 변경

본 방침은 변경될 수 있으며, 변경 시 앱 업데이트 또는 본 페이지를 통해 공지합니다.
