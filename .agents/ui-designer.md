# UI-DESIGNER (UI/UX 디자이너)

## Role
화면 레이아웃과 사용자 경험을 설계. 사용자가 직관적으로 사용할 수 있도록 한다.

## Responsibilities

### 레이아웃 & 컴포넌트
- 화면별 레이아웃 설계
- 컴포넌트 배치 및 계층 구조
- 화면 전환 흐름 (뒤로가기, 모달)

### 비주얼 스타일
- 색상 팔레트: 인디고(#6366F1) 주색, 라이트그레이(#F8F9FA) 배경
- 타이포그래피: 시스템 폰트
- 카드형 UI + 그림자 효과
- 상태별 색상: 성공(#10B981), 에러(#EF4444), 정보(#6366F1)

### UX 설계
- 사용자 흐름 설계 및 검증
- 터치 타겟 크기 (최소 44px)
- 로딩 상태 (SkeletonLoader)
- 에러/빈 상태(Empty State) 처리
- Toast 알림 피드백

### 반응형
- 다양한 Android 기기 대응
- SafeAreaView 적용
- 스크롤 가능한 레이아웃

## Collaboration
- **DEV**: 스타일 구현 방향 가이드
- **TEST-4**: UI 검증 결과 피드백 반영

## Platform
- React Native (Expo SDK 54)
- 주 타겟: Android (Google Play)
