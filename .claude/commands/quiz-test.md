퀴즈 기능을 Puppeteer MCP로 E2E 테스트해줘.

## 테스트 환경
- Expo 웹 앱: http://localhost:8081
- Puppeteer MCP 사용

## 사전 준비
1. `puppeteer_launch`로 브라우저 실행
2. 앱 접속 후 테스트용 단어 5개 등록 (없을 경우)

## 테스트 시나리오

### 1. 퀴즈 설정 화면
- [ ] 단어 수 선택 (5, 10, 20, 전체)
- [ ] 출제 방식 선택 (랜덤, 최근, 취약)
- [ ] 출제 방향 선택 (단어→뜻, 뜻→단어)
- [ ] 단어 부족 시 경고 메시지

### 2. 퀴즈 진행
- [ ] 문제 번호 표시 (1/5, 2/5...)
- [ ] 답 입력 후 "다음" 버튼 동작
- [ ] 마지막 문제에서 "완료" 버튼 표시
- [ ] 정답/오답 피드백 표시 (1.5초)
- [ ] 빈 답 제출 시 처리
- [ ] 중도 종료 시 확인 다이얼로그

### 3. 퀴즈 결과
- [ ] 정답률 표시
- [ ] 전면 광고 표시 여부
- [ ] 오답 목록 표시
- [ ] "틀린 문제만 다시 풀기" 버튼 동작
- [ ] 홈으로 돌아가기

### 4. 엣지 케이스
- [ ] 전체 정답 (100%) 시 결과 화면
- [ ] 전체 오답 (0%) 시 결과 화면
- [ ] 단어 1개로 퀴즈
- [ ] 뜻→단어 모드에서 영어 키보드 힌트

## 결과 보고

```
## 퀴즈 테스트 결과
- 총 테스트: N개
- PASS: N개
- FAIL: N개
- 상세: 각 항목별 결과
```

FAIL 항목은 소스 코드를 읽고 원인 분석 후 수정 방안을 제안해.
